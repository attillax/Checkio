Медиана - это числовое значение, которое делит сортированый массив чисел на большую и меньшую половины. В сортированом массиве с нечетным числом элементов медиана - это число в середине массива. Для массива с четным числом элементов, где нет одного элемента точно посередине, медиана - это среднее значение двух чисел, находящихся в середине массива. 

Дано:
Непустой массив натуральных чисел. 

Найти: 
Медиана данного массива.

Входные данные: 
Массив как список (list) чисел (int).

Выходные данные: 
Медиана как число (int, float).

Примеры:
1 checkio([1, 2, 3, 4, 5]) == 3
2 checkio([3, 1, 2, 5, 3]) == 3
3 checkio([1, 300, 2, 200, 1]) == 2
4 checkio([3, 6, 20, 99, 10, 15]) == 12.5

Алгоритм:
1. Отсортировать входной массив.
2. Для нечетного кол-ва элементов медианой является срединное значение
3. Для четного - среднее арифметическое двух срединных элементов
